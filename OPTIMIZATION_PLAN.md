# 矿大课表优化方案

## 设计依据

- Windows 11 小组件强调“扫一眼即可理解”，小组件应提供高频、简短的信息，并把复杂操作留给主应用：<https://learn.microsoft.com/windows/apps/design/widgets/>
- ClassIsland 的桌面课表实践包含上课进度、组件缩放、背景透明度、窗口层级、位置调整与鼠标穿透：<https://docs.classisland.tech/app/basic.html>
- Power Planner 和 MyStudyLife 将下一课程、提醒、任务与成绩统计作为学生规划软件的核心能力：<https://powerplanner.net/>、<https://mystudylife.com/tour/>

## 本轮实现（P0）

1. 桌面组件独立设置
   - 紧凑、标准、宽屏三种尺寸。
   - 舒适、紧凑两种信息密度。
   - 独立面板和课程卡片透明度。
   - 锁定位置、鼠标穿透、始终置顶。
   - 是否隐藏已经结束的课程。
   - 今天无课时是否自动预览下一教学日。
2. 一眼可读的课程状态
   - 上一日、今天、下一日导航。
   - 当前课程进度条。
   - 下一节课倒计时。
   - 明确区分已结束、进行中和即将开始。
3. 本地课前提醒
   - 可选择提前 5、10、15、20 或 30 分钟发送 Windows 通知。
   - 只依据本机缓存课表工作，不上传数据。
4. 桌面稳定性
   - 继续记忆组件位置和自定义尺寸。
   - 尺寸预设变化时自动调整到合理大小。
   - 鼠标穿透可从系统托盘随时关闭，避免窗口无法操作。

## 后续候选（P1）

- 作业、考试与截止日期：与课程关联，形成“今天/本周待办”。
- 导出 iCalendar（ICS）：同步到 Outlook、手机日历等系统。
- 临时调课与手工课程：在教务课表上叠加本地修改，不改动原始缓存。
- 成绩模拟器：按课程权重预测期末成绩、目标绩点和重修影响。
- 数据备份与恢复：导出不含密码的课表、成绩和个性化设置。
- 网络诊断页：区分校园网不可达、教务维护、登录过期和接口格式变化。

## 暂不优先（P2）

- 云账号和跨设备同步：会扩大隐私、运维与服务器成本。
- 内置 VPN：不符合当前“仅校园网直连”的产品边界。
- 复杂插件系统：当前用户规模和功能复杂度尚不需要。
