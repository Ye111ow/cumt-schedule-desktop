# 更新日志

## v1.0.1 — 2026-07-18

### 成绩读取优化

- 单次按教务系统页面允许的 5000 条读取成绩，并在存在多页时自动继续读取；
- 成绩分项导出未覆盖课程时，自动尝试成绩构成查询接口补全；
- 增强课程名称、全角/半角符号、教学班编号后缀与学分的匹配；
- 成绩页面显示分项覆盖数量，并区分“未公布分项”和“读取失败”。

## v1.0.0 — 2026-07-18

首个公开版本。

### 主要功能

- 中国矿业大学正方教务账号、密码与图片验证码登录；
- 主动导入课表与成绩，启动时只读取本地缓存；
- 7 天周课表和独立今日桌面组件；
- 今日无课或非教学周时保持空白，不自动展示其他日期课程；
- 成绩分项、搜索、筛选、排序、加权成绩和加权绩点；
- 自定义背景、透明度、主题色和明亮/深色模式；
- 桌面固定、置顶、鼠标穿透、系统托盘和课前提醒；
- Windows `safeStorage` 本地加密保存可选凭据。

### 运行要求

- Windows 11 x64；
- 查询或重新导入时需要连接中国矿业大学校园网。
